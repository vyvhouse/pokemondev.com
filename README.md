# ULWmon Landing Page

Static Korean event landing page for **ULWmon : 최고의 포켓몬 트레이너는 어떤 하네스인가?**

## Run locally

```bash
python3 -m http.server 4173
```

Then open <http://localhost:4173>.

## Luma embed

Replace the placeholder inside `#register .luma-embed` in `index.html` with the real Luma embed snippet when the event URL/embed code is available.
